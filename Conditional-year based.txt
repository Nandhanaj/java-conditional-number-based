Q1. Java Program to check whether a year is a leap year or not
import java.util.Scanner;
public class q1 {
  public static void main(String[]args){
    Scanner sc=new Scanner(System.in);
    int yr=sc.nextInt();
    if((yr%4==0 && yr%400==0)||(yr%100!=0)){
      System.out.println("leap year");
    }
    else{
      System.out.println("Not a leap year");
    }

    
  }
}
input:2025               input:1900
output:leap year         output:not a leap year

Q5.Write a Java Program to check whether a given year and month combination is valid and print the number of days in that month (consider leap years for February).
import java.util.Scanner;
public class q1 {
  public static void main(String[]args){
    Scanner sc=new Scanner(System.in);
    int yr=sc.nextInt();
    int mon=sc.nextInt();
    if(yr<=0||mon<1||mon>12){
      System.out.println("Invalid year or month");
    }
    int days=0;
    switch(mon){
      case 1: case 3:case 5:case 7:case 8:case 10:case 12:
        days=31;
        break;
      case 4:case 6:case 9:case 11:
        days=30;
        break;
      case 2:
        if((yr%4==0 &&yr%100!=0)||yr%400==0)
          days=29;
        else
          days=28;
        break;
    }
    System.out.println(days);
  }
}
input:2024  2       input:2025  2
output:29           output:28
